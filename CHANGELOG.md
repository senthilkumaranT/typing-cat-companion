# Changelog

All notable changes to the "typing-cat" extension will be documented in this file.

## [0.1.0] - 2026-06-23

### Added
- Initial release of **Typing Cat Companion**.
- Inline cat decoration rendered at the end of the active coding line.
- 4-frame running animation triggered by typing changes.
- 2-frame sleeping animation triggered by typing inactivity.
- Configuration options for enabling/disabling, idle timeout, animation speed, and cat color scheme (`orange`, `black`, `white`, `gray`).
